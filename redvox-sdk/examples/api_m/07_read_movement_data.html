Reading and Analyzing Movement Detection Data¶

API M enabled clients now have the ability to detect movement onboard when one or both of the gyroscope and accelerometer sensors are enabled.

Movement data is stored in packets as part of an EventStream.

The SDK provides methods for converting the EventStream into a more friendly MovementEventStream.

Encapsulating the Movement Data¶

Let's start by create a class to hold the movement data from multiple packets.

@dataclass
class MovementData:
    """
    Encapsulates movement data from multiple packets.
    """
    gyroscope_ts: np.ndarray = np.array([])
    gyroscope_samples_x: np.ndarray = np.array([])
    gyroscope_samples_y: np.ndarray = np.array([])
    gyroscope_samples_z: np.ndarray = np.array([])
    accel_ts: np.ndarray = np.array([])
    accel_samples_x: np.ndarray = np.array([])
    accel_samples_y: np.ndarray = np.array([])
    accel_samples_z: np.ndarray = np.array([])
    movement_event_stream: MovementEventStream = MovementEventStream("Movement", [])

Next, let's write a function that given a list of packets from a single station, extract and concatenate the relevant movement data. This include any gyroscope timestamps and samples, accelerometer timestamps and samples, and the station derived movement events themselves.

You'll note that MovementEventStream.from_event_stream is used to convert from the generic 'EventStream' into an easier to use 'MovementEventStream'.

def extract_movement_data(packets: List['WrappedRedvoxPacketM']) -> MovementData:
    """
    Extracts and concatenates movement data from a list of packets.
    :param packets: Packets to extract movement data from.
    :return: The encapsulated movement data.
    """

    def reduce_fn(acc: MovementData, packet: 'WrappedRedvoxPacketM') -> MovementData:
        """
        This function is passed to reduce and provides the role of accumulating movement data from packet and storing
        it in acc.
        :param acc: The accumulator.
        :param packet: The packet.
        :return: The updated movement data (the accumulator).
        """
        sensors: 'Sensors' = packet.get_sensors()

        # Extract gyroscope timestamps and X, Y, Z samples if gyroscope exists
        gyro: Optional[Xyz] = sensors.get_gyroscope()
        if gyro is not None:
            acc.gyroscope_ts = np.concatenate([acc.gyroscope_ts, gyro.get_timestamps().get_timestamps()])
            acc.gyroscope_samples_x = np.concatenate([acc.gyroscope_samples_x, gyro.get_x_samples().get_values()])
            acc.gyroscope_samples_y = np.concatenate([acc.gyroscope_samples_y, gyro.get_y_samples().get_values()])
            acc.gyroscope_samples_z = np.concatenate([acc.gyroscope_samples_z, gyro.get_z_samples().get_values()])

        # Extract accelerometer timestamps and X, Y, Z samples if accelerometer exists
        accel: Optional[Xyz] = sensors.get_accelerometer()
        if accel is not None:
            acc.accel_ts = np.concatenate([acc.accel_ts, accel.get_timestamps().get_timestamps()])
            acc.accel_samples_x = np.concatenate([acc.accel_samples_x, accel.get_x_samples().get_values()])
            acc.accel_samples_y = np.concatenate([acc.accel_samples_y, accel.get_y_samples().get_values()])
            acc.accel_samples_z = np.concatenate([acc.accel_samples_z, accel.get_z_samples().get_values()])

        # Extract the derived EventStreams and convert them into MovementEventStreams.
        event_streams: List['EventStream'] = packet.get_event_streams().get_values()
        if len(event_streams) == 1:
            movement_event_stream: MovementEventStream = MovementEventStream.from_event_stream(event_streams[0])
            acc.movement_event_stream.movement_events.extend(movement_event_stream.movement_events)

        return acc

    return reduce(reduce_fn, packets, MovementData())

Plotting Movement Data¶

Now that we have an understanding of what the derived movement metadata looks like, let's use this information to visualize movements alongside the actual gyroscope and accelerometer data. Let's make a figure that plots the power of the signal as well as derived movements that span the signal and represent the mean movement magnitude.

Representing movement on the plot is a little tricky at first glance. We have the start and end time of the movement, and do not want non-movement to be shown at all. To achieve this, we'll use a little trick that matplotlib does not connect series that have NaNs in-between.

Let's write a function that takes a MovementEventStream and converts it into a matplotlib friendly series of datetimes for the x-axis and a numpy array for the y-axis. We will use the movement's mean magnitude as y-values.

def extract_movement_series(movement_event_stream: MovementEventStream,
                            channel: MovementChannel) -> Tuple[List[datetime.datetime], np.ndarray]:
    """
    Given an MovementEventStream and a specified channel, transform the movement data into a list of datetimes and a
    numpy array representing the mean magnitude of the movement. This function inserts NaNs in-between movement so that
    the series is broken up when displayed by matplotlib.
    :param movement_event_stream: A movement event stream to extract series information from.
    :param channel: Channel to extract movement data from.
    :return: A tuple of (datetimes, y-values)
    """
    dts: List[datetime.datetime] = []
    samples: List[float] = []

    for movement_event in movement_event_stream.events_by_channel(channel):
        dts.extend([
            movement_event.movement_start_dt(),
            movement_event.movement_end_dt(),
            # Insert a timestamp directly after the movement end to associate with a nan
            movement_event.movement_end_dt() + datetime.timedelta(seconds=.0001)
        ])
        samples.extend([
            movement_event.magnitude_mean,
            movement_event.magnitude_mean,
            # Insert a nan to break up series in the plot
            float("nan")
        ])

    return dts, np.array(samples)

def plot_movement(movement_data: MovementData, station_id: str) -> None:
    """
    Plot the square of the sensor channels against the mean magnitude of detected movement.
    :param movement_data: Encapsulated movement data.
    :param station_id: Station that is being plotted.
    """

    # Setup the plot
    fig: plt.Figure
    axes: List[plt.Axes]
    fig, axes = plt.subplots(3, 2, figsize=(16, 9))
    fig.subplots_adjust(hspace=0.5)

    gyro_x_ax: plt.Axes = axes[0][0]
    gyro_y_ax: plt.Axes = axes[1][0]
    gyro_z_ax: plt.Axes = axes[2][0]

    accel_x_ax: plt.Axes = axes[0][1]
    accel_y_ax: plt.Axes = axes[1][1]
    accel_z_ax: plt.Axes = axes[2][1]

    # Convert timestamps to datetimes for gyro and accel samples
    gyro_dts: List[datetime.datetime] = list(map(datetime_from_epoch_microseconds_utc, movement_data.gyroscope_ts))
    accel_dts: List[datetime.datetime] = list(map(datetime_from_epoch_microseconds_utc, movement_data.accel_ts))

    # Convert the movement data into something matplotlib can work with
    (gyro_x_movement_dts, gyro_x_movement_samples) = extract_movement_series(movement_data.movement_event_stream,
                                                                             MovementChannel.GYROSCOPE_X)
    (gyro_y_movement_dts, gyro_y_movement_samples) = extract_movement_series(movement_data.movement_event_stream,
                                                                             MovementChannel.GYROSCOPE_Y)
    (gyro_z_movement_dts, gyro_z_movement_samples) = extract_movement_series(movement_data.movement_event_stream,
                                                                             MovementChannel.GYROSCOPE_Z)

    (accel_x_movement_dts, accel_x_movement_samples) = extract_movement_series(movement_data.movement_event_stream,
                                                                               MovementChannel.ACCELEROMETER_X)
    (accel_y_movement_dts, accel_y_movement_samples) = extract_movement_series(movement_data.movement_event_stream,
                                                                               MovementChannel.ACCELEROMETER_Y)
    (accel_z_movement_dts, accel_z_movement_samples) = extract_movement_series(movement_data.movement_event_stream,
                                                                               MovementChannel.ACCELEROMETER_Z)

    gyro_x_ax.plot(gyro_dts, movement_data.gyroscope_samples_x * movement_data.gyroscope_samples_x, label="Samples")
    gyro_x_ax.plot(gyro_x_movement_dts,
                   gyro_x_movement_samples,
                   color="red",
                   label="Detected Movement (mean magnitude)")
    gyro_x_ax.set_title("Gyroscope X")
    gyro_x_ax.set_ylabel("$(rad/s)^2$")
    gyro_x_ax.set_xlabel("Date/Time (UTC)")
    gyro_x_ax.legend()

    gyro_y_ax.plot(gyro_dts, movement_data.gyroscope_samples_y * movement_data.gyroscope_samples_y, label="Samples")
    gyro_y_ax.plot(gyro_y_movement_dts,
                   gyro_y_movement_samples,
                   color="red",
                   label="Detected Movement (mean magnitude)")
    gyro_y_ax.set_title("Gyroscope Y")
    gyro_y_ax.set_ylabel("$(rad/s)^2$")
    gyro_y_ax.set_xlabel("Date/Time (UTC)")
    gyro_y_ax.legend()

    gyro_z_ax.plot(gyro_dts, movement_data.gyroscope_samples_z * movement_data.gyroscope_samples_z, label="Samples")
    gyro_z_ax.plot(gyro_z_movement_dts,
                   gyro_z_movement_samples,
                   color="red",
                   label="Detected Movement (mean magnitude)")
    gyro_z_ax.set_title("Gyroscope Z")
    gyro_z_ax.set_ylabel("$(rad/s)^2$")
    gyro_z_ax.set_xlabel("Date/Time (UTC)")
    gyro_z_ax.legend()

    accel_x_ax.plot(accel_dts, movement_data.accel_samples_x * movement_data.accel_samples_x, label="Samples")
    accel_x_ax.plot(accel_x_movement_dts,
                    accel_x_movement_samples,
                    color="red",
                    label="Detected Movement (mean magnitude)")
    accel_x_ax.set_title("Accelerometer X")
    accel_x_ax.set_ylabel("$(m/s^2)^2$")
    accel_x_ax.set_xlabel("Date/Time (UTC)")
    accel_x_ax.legend()

    accel_y_ax.plot(accel_dts, movement_data.accel_samples_y * movement_data.accel_samples_y, label="Samples")
    accel_y_ax.plot(accel_y_movement_dts,
                    accel_y_movement_samples,
                    color="red",
                    label="Detected Movement (mean magnitude)")
    accel_y_ax.set_title("Accelerometer Y")
    accel_y_ax.set_ylabel("$(m/s^2)^2$")
    accel_y_ax.set_xlabel("Date/Time (UTC)")
    accel_y_ax.legend()

    accel_z_ax.plot(accel_dts, movement_data.accel_samples_z * movement_data.accel_samples_z, label="Samples")
    accel_z_ax.plot(accel_z_movement_dts,
                    accel_z_movement_samples,
                    color="red",
                    label="Detected Movement (mean magnitude)")
    accel_z_ax.set_title("Accelerometer Z")
    accel_z_ax.set_ylabel("$(m/s^2)^2$")
    accel_z_ax.set_xlabel("Date/Time (UTC)")
    accel_z_ax.legend()

    fig.suptitle(f"Detected Movement for {station_id} @ "
                 f"{movement_data.movement_event_stream.movement_events[0].movement_start_dt()}")

    fig.show()
